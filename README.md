## Project Overview
This is a responsive, single-page personal portfolio built using **HTML5 and CSS3**. 
It showcases my skills in front-end development, modern design principles, and responsive layout techniques (using CSS Flexbox and Media Queries).

## Key Features
* Modern, dark-themed design (Midnight Blue and Electric Cyan).
* Fully responsive layout for mobile, tablet, and desktop screens.
* Smooth scrolling navigation.
* Contact form placeholder.

## How to Run Locally
1. Clone or download this repository.
2. Open the `index.html` file in any web browser (Chrome, Firefox, etc.).
